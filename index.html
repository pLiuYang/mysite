Vodoke Mate

Overview:

Vodoke Mate is web portal build upon Spring Boot framework that allows users to store/update information about tenants and set-top-boxs. Set-top-boxs will use the REST API to get their tenant-id. License server will get the list of tenants and set-top-boxs mac-address from Vodoke Mate.

My Contribution:

  • Set up project using Spring Boot
  • Add CRUD functions
  • Create REST API
Year: 2016
